# COVID-19 Global Vaccination Analysis & SARIMA Forecasting

COVID-19 global vaccination analysis & SARIMA forecasting. EDA + time-series modeling + Streamlit dashboard.

## Overview
This repository contains the data analysis, SARIMA forecasting, and Streamlit dashboard code for the COVID-19 Global Vaccination Analysis project.

## How to run
1. Install dependencies: `pip install -r requirements.txt`
2. Run Streamlit app: `streamlit run app.py`
